# Project Demo Flow

## 1. Project Introduction
AI-powered system that collects infrastructure project news, extracts key information, scores opportunities, and visualizes insights.

## 2. Dashboard Overview
Show KPIs:
- Total Projects
- High Opportunity Projects
- Agencies
- Average Score

## 3. Filters & Search
Demonstrate project type, agency, and project search filters.

## 4. Opportunity Recommendation Engine
Search for keywords and display recommended opportunities.

## 5. Agency Influence Map
Show which agencies are driving the most projects.

## 6. Location Analytics
Analyze infrastructure opportunities by location.

## 7. Sector Opportunity Analysis
Compare sectors using opportunity scores.

## 8. Project Explorer
Review detailed project information.

## 9. PDF Report Generation
Generate and download executive report.

## 10. Future Enhancements
- Automated alerts
- Tender integration
- Competitor intelligence
- ML-based opportunity prediction
